# Alexandria

The check in and check out system of choice for ACM@UIUC.

### Version

Still in development!

### Installation

```sh
$ git clone https://github.com/mathur/alexandria.git alexandria
$ cd alexandria
$ . venv/bin/activate
$ pip install -r requirements.txt
```

### Running

```sh
$ . venv/bin/activate
$ python app.py
```
